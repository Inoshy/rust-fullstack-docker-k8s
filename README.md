## Build
>      $ cd app
>      $ cd npm run build
>      $ cd ../
>      $ cargo build

## Run
>      $ cargo run

## Deploy to K8s on GKE
>      $ ./scripts/build.sh
>      $ ./scripts/deploy-gke.sh
